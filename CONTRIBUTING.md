# Contributing to Awesome AI Game Dev

Thank you for contributing! This list is community-maintained and relies on submissions like yours.

## How to Add a Tool

1. Fork this repository.
2. Add your tool to the relevant section in `README.md` or the appropriate `pipeline/` file.
3. Submit a pull request with a clear description.

Alternatively, open an issue using the [Add Tool template](.github/ISSUE_TEMPLATE/add-tool.yml).

## Entry Format

```
- [Tool Name](https://link) - One-sentence factual description of what the tool does.
```

## Guidelines

- **Factual descriptions only** — avoid subjective claims like "best", "revolutionary", or "most powerful".
- **Working links** — verify the URL is reachable before submitting.
- **Correct category** — place the tool in the pipeline stage it primarily addresses.
- **No duplicates** — search existing entries before adding.
- **Open source or publicly accessible** — tools must be accessible to the community.

## What We Don't Accept

- Tools that are no longer maintained or have dead links.
- Submissions that are purely promotional with no clear utility.
- Duplicate entries.

## Review Process

Maintainers will review PRs within a few days. We may ask for clarifications or suggest a different category.
